Yahtzee V1 - Overview

This is an online Yahtzee dice game for two players. It is created as a static website using HTML and CSS, with all functionality provided within the browser using javascript.
The user can roll their dice up to three times each turn, holding as many dice as they choose between rerolls by clicking the dice. When they are ready, they click the score section they want to use their dice for - green sections being valid, red sections being invalid (and thus scoring 0). At the end of the game, scores are calculated.
